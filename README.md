# Python Number Guessing Game 🎯

This is a simple **Number Guessing Game written in Python**.
The program randomly selects a number between **1 and 100**, and the player must guess the correct number based on hints provided by the program.

## 💻 Example Output

```
Welcome to the Number Guessing Game!
Guess a number between 1 and 100

Enter your guess: 45
Too low! Try again.

Enter your guess: 70
Too high! Try again.

Enter your guess: 63
Congratulations! You guessed the number in 3 attempts.
```

## 🧠 Concepts Used

* Python basics
* Random module
* Loops
* Conditional statements
* User input handling



## 📚 Purpose

This project is designed for **beginners learning Python programming** and helps practice basic programming concepts.
