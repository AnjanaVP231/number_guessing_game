# Python Number Guessing Game 🎯

This is a simple **Number Guessing Game written in Python**.
The program randomly selects a number between **1 and 100**, and the player must guess the correct number based on hints provided by the program.

## 🚀 Features

* Random number generation using Python
* User input from the terminal
* Hints for each guess (Too High / Too Low)
* Tracks the number of attempts
* Simple and beginner-friendly project

## 🕹️ How the Game Works

1. The program generates a random number between **1 and 100**.
2. The player enters a guess.
3. The program compares the guess with the secret number.
4. The program displays:

   * **Too High** if the guess is larger than the number.
   * **Too Low** if the guess is smaller than the number.
5. The game continues until the player guesses the correct number.
